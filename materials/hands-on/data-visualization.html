Hands-on - Activity 3

Data visualization with Quarto and Observable JavaScript

Objective: Create a Quarto document that downloads sample data from a URL and creates interactive visualizations using pure Observable JavaScript.

Step 1: Create a New Quarto Document

Create a file called penguins-ojs.qmd with this basic structure:

---
title: "Penguins Data Analysis with OJS"
format: html
---

Step 2: Download or Import Data with OJS

Add this OJS code block to fetch the penguins data directly from the web:

```{ojs}
//| label: load-data
// Fetch the penguins data from GitHub Gist
penguins_data = FileAttachment(("https://gist.githubusercontent.com/slopp/ce3b90b9168f2f921784de84fa445651/raw/4ecf3041f0ed4913e7c230758733948bc561f434/penguins.csv")).csv({typed: true})
```
Note

You can also download the penguins data set and worl locally.

Step 3: Create Data Preview

Add this OJS block to display the raw data in a table:

```{ojs}
//| label: preview-data
//| echo: false
// Display interactive data table

Inputs.table(penguins_data, {
  label: "Penguins Dataset",
  rows: 10,
  sort: "species"
})
```

Step 4: Add Interactive Controls (Optional Extension)

For more advanced exploration, add this interactive visualization:

```{ojs}
//| label: plot-controls
//| echo: false
// Create controls for the interactive plot
viewof x_var = Inputs.select(
  ["bill_length_mm", "bill_depth_mm", "flipper_length_mm", "body_mass_g"],
  {label: "X-axis", value: "bill_length_mm"}
)

viewof y_var = Inputs.select(
  ["bill_length_mm", "bill_depth_mm", "flipper_length_mm", "body_mass_g"], 
  {label: "Y-axis", value: "bill_depth_mm"}
)

viewof color_var = Inputs.select(
  ["species", "island", "sex"],
  {label: "Color by", value: "species"}
)
```
```{ojs}
//| label: interactive-plot
//| echo: false
// Create the interactive scatter plot
Plot.plot({
  title: `Penguin Measurements`,
  marks: [
    Plot.dot(penguins_data, {
      x: x_var,       // Now correctly references the reactive value
      y: y_var,       // Now correctly references the reactive value
      fill: color_var, // Now correctly references the reactive value
      stroke: "black",
      strokeWidth: 0.5,
      r: 5,
      tip: true
    })
  ],
  color: {
    legend: true
  },
  x: {
    grid: true,
```

## Summary

This analysis demonstrates:

- **Data Loading**: Using `FileAttachment()` to load external CSV data
- **Interactive Tables**: Using `Inputs.table()` to explore data
- **Interactive Visualizations**: Using Observable Plot with dynamic controls
- **Reactive Programming**: OJS automatically updates visualizations when controls change

Try changing the dropdown menus above to explore different aspects of the penguins dataset!

All of this is accomplished using **pure Observable JavaScript** without requiring R or Python.

Key Learning Points

  • FileAttachment: Loads external data files in OJS
  • Inputs.table(): Creates interactive data tables
  • Plot.dot(): Creates scatter plots with Observable Plot
  • d3.rollup(): Calculates summary statistics
  • viewof: Creates reactive variables that update visualizations

This approach works entirely in the browser and requires no additional software beyond Quarto!
